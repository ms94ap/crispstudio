# TASK CREATOR

## GET STARTED

**1). Clone Repo**

**2) Now run installers**

  ```
  $ bundle install
  $ yarn install
  ```
 **3) Run Server**
 
 ```
 $ rails s
 ```

 ### For Development

 Material UI is pre installed and to implement import in a React Component as such :

 ```
 import { Button } from '@material-ui/core';

 ```

 For more details visit [Material UI](https://material-ui.com/)
